# My Product
## 소개
product 정보를 관리하는 msa 서비스르 구현한다. 
`스프링으로 하는 마이크로서비스` 책 내용을 기반으로 한다. 

## 서비스
1. Product 서버
2. Review 서버
3. Recommendation 서버
4. bff 서버

## 사용 기술
기술
- OAS(open api spec.)
- event driven (kafka)
- circuit breaker (resilience4j)
- issue tracking (slueth)
- reactive (webflux, coroutine)
